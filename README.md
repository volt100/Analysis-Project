# Project: Attrition Data Analysis
This project analyzes the attrition dataset using Python.

## How to run:
1. Install requirements: `pip install pandas scikit-learn matplotlib seaborn imblearn`
2. Run `main.py` (or your main script name).

## Scripts included:
* `c306_data_understanding_3.py`: EDA and Visualizations.
* `c306_data_preprocess_3.py`: Data cleaning and balancing (SMOTE).
